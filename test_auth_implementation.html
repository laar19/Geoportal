⚙️ Variable de Control

La variable ENABLE_KEYCLOAK_AUTH controla el sistema:

  • ENABLE_KEYCLOAK_AUTH=true: Sistema auth habilitado
  • ENABLE_KEYCLOAK_AUTH=false: Sistema auth deshabilitado (testing rápido)
